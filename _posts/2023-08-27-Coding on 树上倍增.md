#### 树上倍增

基环树、图排序

适用场景：按照题目要求，数组会被分割成若干段，且每一段的判断/处理逻辑是一样的。

ps. 为了不让代码过于 ugly



#### Template

```
i, n = 0, len(nums)
while i < n:
    start = i
    while i < n and ...:
        i += 1
    # 从 start 到 i-1 是一段
    # 下一段从 i 开始，无需 i+=1
```
